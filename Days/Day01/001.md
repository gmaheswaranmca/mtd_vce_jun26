## Problem: Pow(x, n)

Implement `x^n` efficiently using binary exponentiation.

**Asked By:** Amazon, Google, Meta, Microsoft, Apple
**Difficulty:** Medium

---

# Solution Patterns

## Pattern 1: Brute Force Multiplication

### Idea

Multiply `x` by itself `n` times.

### Complexity

* Time: **O(n)**
* Space: **O(1)**

### Code

```python
def myPow(x: float, n: int) -> float:
    if n < 0:
        x = 1 / x
        n = -n

    result = 1.0

    for _ in range(n):
        result *= x

    return result
```

### Drawback

Very slow for large values of `n`.

---

# Pattern 2: Recursive Binary Exponentiation (Divide & Conquer)

### Idea

Use:

* xⁿ = (x²)ⁿᐟ² when n is even
* xⁿ = x × (x²)ⁿᐟ² when n is odd

Reduce exponent by half every step.

### Complexity

* Time: **O(log n)**
* Space: **O(log n)** (recursive stack)

### Code

```python
def myPow(x: float, n: int) -> float:

    def power(base, exp):
        if exp == 0:
            return 1

        half = power(base, exp // 2)

        if exp % 2 == 0:
            return half * half
        else:
            return base * half * half

    if n < 0:
        x = 1 / x
        n = -n

    return power(x, n)
```

# Pattern 3: Iterative Binary Exponentiation (Best Pattern)

### Idea

Observe binary representation of exponent.

Example:

```text
13 = 1101₂

2¹³
= 2⁸ × 2⁴ × 2¹
```

Whenever the current bit is 1:

```python
result *= current_power
```

Then:

```python
current_power *= current_power
n >>= 1
```

### Complexity

* Time: **O(log n)**
* Space: **O(1)**

### Code

```python
def myPow(x: float, n: int) -> float:

    if n < 0:
        x = 1 / x
        n = -n

    result = 1.0

    while n > 0:

        if (n & 1) == 1:
            result *= x

        x *= x
        n >>= 1

    return result
```

# Optimized Interview Pattern

Use **Iterative Binary Exponentiation**.

Reasons:

1. Reduces exponent by half every iteration.
2. Handles very large exponents efficiently.
3. No recursion stack.
4. Achieves:

   * **Time:** O(log n)
   * **Space:** O(1)

### Final Recommended Code

```python
def myPow(x: float, n: int) -> float:
    if n < 0:
        x = 1 / x
        n = -n

    result = 1.0

    while n:
        if n & 1:
            result *= x

        x *= x
        n >>= 1

    return result
```

This is the solution most commonly expected in Amazon, Google, Meta, Microsoft, and Apple interviews.
