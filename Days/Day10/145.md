# LeetCode 145 - Binary Tree Postorder Traversal

## Category

Binary Tree, DFS, Recursion, Stack

---

# Description

Given the `root` of a binary tree, return its **postorder traversal**.

Postorder traversal visits nodes in the order:

```text
Left → Right → Root
```

Return the traversal as a list.

---

## Sample Input

```text
        1
         \
          2
         /
        3
```

## Sample Output

```text
[3,2,1]
```

---

## Explanation

Traversal order:

```text
Go Right → 2

Go Left → 3

Visit 3

Visit 2

Visit 1
```

Result:

```text
[3,2,1]
```

---

# Idea to Solve

For every node:

1. Traverse left subtree.
2. Traverse right subtree.
3. Visit current node.

Postorder is commonly used for:

* Tree deletion
* Expression tree evaluation
* Computing subtree information
* Dynamic Programming on Trees

# Algorithm (Recursive)

1. If node is NULL, return.
2. Traverse left subtree.
3. Traverse right subtree.
4. Visit node.
5. Return result.

---

# Pseudocode

```text
FUNCTION postorder(node, result)
    IF node IS NULL
        RETURN
    postorder(node.left, result)
    postorder(node.right, result)
    ADD node.value TO result
END FUNCTION
FUNCTION postorderTraversal(root)
    result = []
    postorder(root, result)
    RETURN result
END FUNCTION
```

# Iterative Solution (Single Stack + Reverse)

## Key Observation

Postorder:

```text
Left → Right → Root
```

If we generate:

```text
Root → Right → Left
```

and reverse it,

we get:

```text
Left → Right → Root
```

Exactly postorder.

# Algorithm

1. Push root.
2. Pop node.
3. Store node value.
4. Push left child.
5. Push right child.
6. Reverse result.

---

# Pseudocode

```text
FUNCTION postorderTraversal(root)

    IF root IS NULL
        RETURN []

    stack = [root]

    result = []

    WHILE stack NOT EMPTY

        node = POP stack

        ADD node.value TO result

        IF node.left EXISTS
            PUSH node.left

        IF node.right EXISTS
            PUSH node.right

    END WHILE

    REVERSE result

    RETURN result

END FUNCTION
```

---

# Python (Iterative)

```python
class Solution:

    def postorderTraversal(self, root):

        if root is None:
            return []

        stack = [root]
        result = []

        while stack:

            node = stack.pop()

            result.append(node.val)

            if node.left:
                stack.append(node.left)

            if node.right:
                stack.append(node.right)

        result.reverse()

        return result
```

# C++ (Iterative)

```cpp
class Solution {
public:

    vector<int> postorderTraversal(TreeNode* root) {

        vector<int> result;

        if(root == nullptr)
            return result;

        stack<TreeNode*> st;

        st.push(root);

        while(!st.empty()) {

            TreeNode* node = st.top();
            st.pop();

            result.push_back(node->val);

            if(node->left)
                st.push(node->left);

            if(node->right)
                st.push(node->right);
        }

        reverse(result.begin(),
                result.end());

        return result;
    }
};
```

# Java (Iterative)

```java
class Solution {

    public List<Integer> postorderTraversal(
            TreeNode root) {

        List<Integer> result =
            new ArrayList<>();

        if(root == null)
            return result;

        Stack<TreeNode> stack =
            new Stack<>();

        stack.push(root);

        while(!stack.isEmpty()) {

            TreeNode node = stack.pop();

            result.add(node.val);

            if(node.left != null)
                stack.push(node.left);

            if(node.right != null)
                stack.push(node.right);
        }

        Collections.reverse(result);

        return result;
    }
}
```

# Traversal Summary

| Traversal   | Order               |
| ----------- | ------------------- |
| Preorder    | Root → Left → Right |
| Inorder     | Left → Root → Right |
| Postorder   | Left → Right → Root |
| Level Order | BFS                 |

Example:

```text
        1
       / \
      2   3
```

Preorder:

```text
[1,2,3]
```

Inorder:

```text
[2,1,3]
```

Postorder:

```text
[2,3,1]
```

Level Order:

```text
[1,2,3]
```

### Iterative Tricks

```text
Preorder:
Push Right then Left

Inorder:
Keep moving Left

Postorder:
Root Right Left + Reverse
```
