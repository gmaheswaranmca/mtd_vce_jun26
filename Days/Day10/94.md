# LeetCode 94 - Binary Tree Inorder Traversal

## Category

Binary Tree, DFS, Recursion, Stack

---

# Description

Given the `root` of a binary tree, return its **inorder traversal**.

Inorder traversal visits nodes in the order:

```text
Left → Root → Right
```

Return the traversal as a list.

---

## Sample Input

```text
        1
         \
          2
         /
        3
```

## Sample Output

```text
[1,3,2]
```

---

## Explanation

Inorder traversal:

```text
Left of 1 = NULL

Visit 1

Go Right → 2

Left of 2 → 3

Visit 3

Visit 2
```

Result:

```text
[1,3,2]
```

---

# Idea to Solve

For every node:

1. Traverse left subtree.
2. Visit current node.
3. Traverse right subtree.

This naturally fits recursion.

# Algorithm (Recursive)

1. If node is NULL, return.
2. Traverse left subtree.
3. Add current node value.
4. Traverse right subtree.
5. Return result.

---

# Pseudocode

```text
FUNCTION inorder(node, result)
    IF node IS NULL
        RETURN
    inorder(node.left, result)
    ADD node.value TO result
    inorder(node.right, result)
END FUNCTION
FUNCTION inorderTraversal(root)
    result = []
    inorder(root, result)
    RETURN result
END FUNCTION
```

---

# Python (Recursive)

```python
class Solution:

    def inorder(self, node, result):

        if node is None:
            return

        self.inorder(node.left, result)

        result.append(node.val)

        self.inorder(node.right, result)

    def inorderTraversal(self, root):

        result = []

        self.inorder(root, result)

        return result
```

# Iterative Solution (Using Stack)

### Idea

Recursion internally uses a call stack.

We can simulate it manually.

Steps:

1. Keep moving left.
2. Push nodes onto stack.
3. When left becomes NULL:

   * Pop node
   * Visit node
   * Move right

---

## Pseudocode

```text
FUNCTION inorderTraversal(root)

    stack = empty

    current = root

    result = []

    WHILE current IS NOT NULL
          OR stack NOT EMPTY

        WHILE current IS NOT NULL

            PUSH current

            current = current.left

        END WHILE

        current = POP stack

        ADD current.value TO result

        current = current.right

    END WHILE

    RETURN result

END FUNCTION
```

---

# Python (Iterative)

```python
class Solution:

    def inorderTraversal(self, root):

        stack = []
        result = []

        current = root

        while current or stack:

            while current:
                stack.append(current)
                current = current.left

            current = stack.pop()

            result.append(current.val)

            current = current.right

        return result
```

# C++ (Iterative)

```cpp
class Solution {
public:

    vector<int> inorderTraversal(TreeNode* root) {

        stack<TreeNode*> st;

        vector<int> result;

        TreeNode* current = root;

        while(current != nullptr ||
              !st.empty()) {

            while(current != nullptr) {

                st.push(current);

                current = current->left;
            }

            current = st.top();
            st.pop();

            result.push_back(current->val);

            current = current->right;
        }

        return result;
    }
};
```

# Java (Iterative)

```java
class Solution {

    public List<Integer> inorderTraversal(
            TreeNode root) {

        Stack<TreeNode> stack =
            new Stack<>();

        List<Integer> result =
            new ArrayList<>();

        TreeNode current = root;

        while(current != null ||
              !stack.isEmpty()) {

            while(current != null) {

                stack.push(current);

                current = current.left;
            }

            current = stack.pop();

            result.add(current.val);

            current = current.right;
        }

        return result;
    }
}
```

# Traversal Cheat Sheet

| Traversal   | Order               |
| ----------- | ------------------- |
| Preorder    | Root → Left → Right |
| Inorder     | Left → Root → Right |
| Postorder   | Left → Right → Root |
| Level Order | BFS                 |

Example:

```text
        1
       / \
      2   3
```

Preorder:

```text
[1,2,3]
```

Inorder:

```text
[2,1,3]
```

Postorder:

```text
[2,3,1]
```

Level Order:

```text
[1,2,3]
```
