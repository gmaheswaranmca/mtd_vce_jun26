# LeetCode 112 - Path Sum

## Category

Binary Tree

## Description

Given the root of a binary tree and an integer `targetSum`, return `true` if the tree has a **root-to-leaf** path such that adding up all the values along the path equals `targetSum`.

A leaf is a node with no left and right children.

---

## Sample Input

```text
Tree:

        5
      /   \
     4     8
    /     / \
   11    13  4
  /  \
 7    2

targetSum = 22
```

## Sample Output

```text
true
```

## Explanation

Root-to-leaf path:

```text
5 → 4 → 11 → 2
```

Sum:

```text
5 + 4 + 11 + 2 = 22
```

Hence return `true`.

---

## Idea to Solve

At every node:

1. Subtract current node value from target.
2. Move to left and right subtree.
3. If a leaf node is reached and remaining target becomes 0, answer is true.

This is a DFS traversal problem.

---

## Algorithm

1. If root is NULL, return false.
2. Reduce target by current node value.
3. If current node is a leaf:

   * Return true if remaining target is 0.
4. Recursively check left subtree.
5. Recursively check right subtree.
6. Return OR of both results.

---

## Pseudocode

```text
FUNCTION hasPathSum(node, target)

    IF node IS NULL
        RETURN FALSE

    target = target - node.value

    IF node.left IS NULL AND node.right IS NULL
        RETURN target == 0

    RETURN
        hasPathSum(node.left, target)
        OR
        hasPathSum(node.right, target)

END FUNCTION
```

---

# Python

```python
class Solution:
    def hasPathSum(self, root, targetSum):
        if root is None:
            return False

        targetSum -= root.val

        if root.left is None and root.right is None:
            return targetSum == 0

        return (
            self.hasPathSum(root.left, targetSum) or
            self.hasPathSum(root.right, targetSum)
        )
```

---

# C++

```cpp
class Solution {
public:
    bool hasPathSum(TreeNode* root, int targetSum) {

        if(root == nullptr)
            return false;

        targetSum -= root->val;

        if(root->left == nullptr && root->right == nullptr)
            return targetSum == 0;

        return hasPathSum(root->left, targetSum) ||
               hasPathSum(root->right, targetSum);
    }
};
```

## Complexity Analysis

### Time Complexity

```text
O(N)
```

Every node is visited once.

### Space Complexity

```text
O(H)
```

Where `H` is tree height due to recursion stack.

* Balanced tree: `O(log N)`
* Skewed tree: `O(N)`
