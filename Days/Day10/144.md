# LeetCode 144 - Binary Tree Preorder Traversal

## Category

Binary Tree, DFS, Recursion, Stack

---

# Description

Given the `root` of a binary tree, return its **preorder traversal**.

Preorder traversal visits nodes in the order:

```text
Root → Left → Right
```

Return the traversal as a list.

---

## Sample Input

```text
        1
         \
          2
         /
        3
```

## Sample Output

```text
[1,2,3]
```

---

## Explanation

Traversal order:

```text
Visit 1

Go Right → 2

Visit 2

Go Left → 3

Visit 3
```

Result:

```text
[1,2,3]
```

---

# Idea to Solve

For every node:

1. Visit current node.
2. Traverse left subtree.
3. Traverse right subtree.

This is naturally implemented using DFS recursion.

# Algorithm (Recursive)

1. If node is NULL, return.
2. Visit node.
3. Traverse left subtree.
4. Traverse right subtree.
5. Return result.

---

# Pseudocode

```text
FUNCTION preorder(node, result
    IF node IS NULL
        RETURN
    ADD node.value TO result
    preorder(node.left, result)
    preorder(node.right, result)
END FUNCTION
FUNCTION preorderTraversal(root)
    result = []
    preorder(root, result)
    RETURN result
END FUNCTION
```

class Solution:

        self.preorder(node.right, result)

# Iterative Solution (Using Stack)

## Key Observation

Preorder:

```text
Root → Left → Right
```

Stack is LIFO.

To process Left before Right:

```text
Push Right first
Push Left second
```

Then Left comes out first.

---

# Algorithm

1. Push root into stack.
2. While stack is not empty:

   * Pop node.
   * Visit node.
   * Push right child.
   * Push left child.
3. Return result.

---

# Pseudocode

```text
FUNCTION preorderTraversal(root)

    IF root IS NULL
        RETURN []

    stack = [root]

    result = []

    WHILE stack NOT EMPTY

        node = POP stack

        ADD node.value TO result

        IF node.right EXISTS
            PUSH node.right

        IF node.left EXISTS
            PUSH node.left

    END WHILE

    RETURN result

END FUNCTION
```

---

# Python (Iterative)

```python
class Solution:

    def preorderTraversal(self, root):

        if root is None:
            return []

        stack = [root]
        result = []

        while stack:

            node = stack.pop()

            result.append(node.val)

            if node.right:
                stack.append(node.right)

            if node.left:
                stack.append(node.left)

        return result
```

---

# C++ (Iterative)

```cpp
class Solution {
public:

    vector<int> preorderTraversal(TreeNode* root) {

        vector<int> result;

        if(root == nullptr)
            return result;

        stack<TreeNode*> st;

        st.push(root);

        while(!st.empty()) {

            TreeNode* node = st.top();
            st.pop();

            result.push_back(node->val);

            if(node->right)
                st.push(node->right);

            if(node->left)
                st.push(node->left);
        }

        return result;
    }
};
```

---

# Java (Iterative)

```java
class Solution {

    public List<Integer> preorderTraversal(
            TreeNode root) {

        List<Integer> result =
            new ArrayList<>();

        if(root == null)
            return result;

        Stack<TreeNode> stack =
            new Stack<>();

        stack.push(root);

        while(!stack.isEmpty()) {

            TreeNode node = stack.pop();

            result.add(node.val);

            if(node.right != null)
                stack.push(node.right);

            if(node.left != null)
                stack.push(node.left);
        }

        return result;
    }
}
```

# Traversal Comparison

| Traversal   | Order               |
| ----------- | ------------------- |
| Preorder    | Root → Left → Right |
| Inorder     | Left → Root → Right |
| Postorder   | Left → Right → Root |
| Level Order | BFS                 |

Example:

```text
        1
       / \
      2   3
```

Preorder:

```text
[1,2,3]
```

Inorder:

```text
[2,1,3]
```

Postorder:

```text
[2,3,1]
```

Level Order:

```text
[1,2,3]
```

### Stack Trick

```text
Preorder:
Root Left Right

Push:
Right first
Left second
```

So that:

```text
Left is processed first
```
