# LeetCode 105 - Construct Binary Tree from Preorder and Inorder Traversal

## Category

Binary Tree, Divide and Conquer, Recursion

---

# Description

Given two integer arrays:

* `preorder`
* `inorder`

Construct and return the binary tree.

You may assume:

* All values are unique.
* The given traversals represent a valid binary tree.

---

## Traversal Properties

### Preorder

```text
Root → Left → Right
```

### Inorder

```text
Left → Root → Right
```

---

## Sample Input

```text
preorder = [3,9,20,15,7]

inorder  = [9,3,15,20,7]
```

---

## Sample Output

```text
        3
       / \
      9  20
         / \
        15  7
```

---

# Why This Works

### Preorder

```text
[3, 9, 20, 15, 7]
 ^
 Root
```

First element is always root.

Root:

```text
3
```

---

### Inorder

```text
[9, 3, 15, 20, 7]
    ^
   Root
```

Everything left of 3 belongs to left subtree.

```text
[9]
```

Everything right belongs to right subtree.

```text
[15,20,7]
```

# Idea to Solve

For every recursive call:

1. Take next preorder element as root.
2. Find root position in inorder.
3. Elements left of root belong to left subtree.
4. Elements right of root belong to right subtree.
5. Recursively construct left and right trees.

To avoid repeatedly searching inorder:

Use a hashmap:

```text
value -> index
```

---

# Algorithm

1. Create hashmap from inorder.
2. Maintain preorder index.
3. Create root using preorder index.
4. Find root position in inorder.
5. Build left subtree.
6. Build right subtree.
7. Return root.

---

# Pseudocode

```text
GLOBAL preIndex = 0

FUNCTION build(preorder, inorderMap,        left, right)
    IF left > right
        RETURN NULL
    value = preorder[preIndex]
    preIndex = preIndex + 1
    root = new Node(value)
    mid = inorderMap[value]
    root.left = build(preorder, inorderMap, left, mid - 1)
    root.right = build(preorder, inorderMap, mid + 1, right)
    RETURN root
END FUNCTION
FUNCTION buildTree(preorder, inorder)
    inorderMap = {}
    STORE all inorder indices
    preIndex = 0
    RETURN build(preorder, inorderMap, 0, length(inorder)-1)
END FUNCTION
```

# Python

```python
class Solution:

    def build(self, preorder, inorderMap, left, right):

        if left > right:
            return None

        value = preorder[self.preIndex]
        self.preIndex += 1

        root = TreeNode(value)

        mid = inorderMap[value]

        root.left = self.build(
            preorder,
            inorderMap,
            left,
            mid - 1
        )

        root.right = self.build(
            preorder,
            inorderMap,
            mid + 1,
            right
        )

        return root

    def buildTree(self, preorder, inorder):

        inorderMap = {}

        for i, value in enumerate(inorder):
            inorderMap[value] = i

        self.preIndex = 0

        return self.build(
            preorder,
            inorderMap,
            0,
            len(inorder) - 1
        )
```

# C++

```cpp
class Solution {
public:

    int preIndex = 0;

    TreeNode* build(
        vector<int>& preorder,
        unordered_map<int,int>& inorderMap,
        int left,
        int right) {

        if(left > right)
            return nullptr;

        int value = preorder[preIndex++];

        TreeNode* root = new TreeNode(value);

        int mid = inorderMap[value];

        root->left =
            build(preorder,
                  inorderMap,
                  left,
                  mid - 1);

        root->right =
            build(preorder,
                  inorderMap,
                  mid + 1,
                  right);

        return root;
    }

    TreeNode* buildTree(
        vector<int>& preorder,
        vector<int>& inorder) {

        unordered_map<int,int> inorderMap;

        for(int i=0;i<inorder.size();i++)
            inorderMap[inorder[i]] = i;

        return build(
            preorder,
            inorderMap,
            0,
            inorder.size()-1
        );
    }
};
```

# Java

```java
class Solution {

    private int preIndex = 0;

    private TreeNode build(
            int[] preorder,
            Map<Integer,Integer> inorderMap,
            int left,
            int right) {

        if(left > right)
            return null;

        int value = preorder[preIndex++];

        TreeNode root = new TreeNode(value);

        int mid = inorderMap.get(value);

        root.left =
            build(preorder,
                  inorderMap,
                  left,
                  mid - 1);

        root.right =
            build(preorder,
                  inorderMap,
                  mid + 1,
                  right);

        return root;
    }

    public TreeNode buildTree(
            int[] preorder,
            int[] inorder) {

        Map<Integer,Integer> inorderMap =
            new HashMap<>();

        for(int i=0;i<inorder.length;i++)
            inorderMap.put(inorder[i], i);

        return build(
            preorder,
            inorderMap,
            0,
            inorder.length - 1
        );
    }
}
```
