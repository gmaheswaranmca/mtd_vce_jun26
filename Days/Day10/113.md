# LeetCode 113 - Path Sum II

## Category

Binary Tree, DFS, Backtracking

---

## Description

Given the root of a binary tree and an integer `targetSum`, return **all root-to-leaf paths** where the sum of the node values in the path equals `targetSum`.

A leaf is a node with no children.

Return the paths in any order.

---

## Sample Input

```text
Tree:

           5
         /   \
        4     8
       /     / \
      11    13  4
     / \       / \
    7   2     5   1

targetSum = 22
```

## Sample Output

```text
[
  [5,4,11,2],
  [5,8,4,5]
]
```

---

## Explanation

Path 1:

```text
5 → 4 → 11 → 2 = 22
```

Path 2:

```text
5 → 8 → 4 → 5 = 22
```

Both are root-to-leaf paths.

---

# Idea to Solve

Unlike Problem 112, we must return **all valid paths**.

While traversing:

1. Add current node to current path.
2. Subtract node value from target.
3. If leaf node and remaining target becomes 0:

   * Store a copy of path.
4. Explore left subtree.
5. Explore right subtree.
6. Remove current node before returning (Backtracking).

---

# Algorithm

1. Create result list.
2. Start DFS from root.
3. Add node value to current path.
4. Reduce target.
5. If leaf and target becomes 0:

   * Save path copy.
6. Recurse left.
7. Recurse right.
8. Remove last element from path.
9. Return result.

---

# Pseudocode

```text
FUNCTION dfs(node, target, path, result)
    IF node IS NULL
        RETURN
    APPEND node.value TO path
    target = target - node.value
    IF node IS LEAF AND target == 0
        ADD COPY(path) TO result
    ELSE
        dfs(node.left, target, path, result)
        dfs(node.right, target, path, result)
    REMOVE LAST ELEMENT FROM path
END FUNCTION
FUNCTION pathSum(root, target)
    result = []
    dfs(root, target, [], result)
    RETURN result
END FUNCTION
```

# Python

```python
class Solution:

    def dfs(self, node, target, path, result):

        if node is None:
            return

        path.append(node.val)

        target -= node.val

        if node.left is None and node.right is None and target == 0:
            result.append(path.copy())
        else:
            self.dfs(node.left, target, path, result)
            self.dfs(node.right, target, path, result)

        path.pop()

    def pathSum(self, root, targetSum):

        result = []

        self.dfs(root, targetSum, [], result)

        return result
```

# C++

class Solution {
public:

    void dfs(TreeNode* node,
             int target,
             vector<int>& path,
             vector<vector<int>>& result) {

        if(node == nullptr)
            return;

        path.push_back(node->val);

        target -= node->val;

        if(node->left == nullptr &&
           node->right == nullptr &&
           target == 0) {

            result.push_back(path);
        }
        else {

            dfs(node->left, target, path, result);
            dfs(node->right, target, path, result);
        }

        path.pop_back();
    }

    vector<vector<int>> pathSum(TreeNode* root,
                                int targetSum) {

        vector<vector<int>> result;
        vector<int> path;

        dfs(root, targetSum, path, result);

        return result;
    }
};

# Java

```java
class Solution {

    private void dfs(TreeNode node,
                     int target,
                     List<Integer> path,
                     List<List<Integer>> result) {

        if(node == null)
            return;

        path.add(node.val);

        target -= node.val;

        if(node.left == null &&
           node.right == null &&
           target == 0) {

            result.add(new ArrayList<>(path));
        }
        else {

            dfs(node.left, target, path, result);
            dfs(node.right, target, path, result);
        }

        path.remove(path.size() - 1);
    }

    public List<List<Integer>> pathSum(TreeNode root,
                                       int targetSum) {

        List<List<Integer>> result = new ArrayList<>();

        dfs(root, targetSum,
            new ArrayList<>(),
            result);

        return result;
    }
}
```

# Complexity Analysis

### Time Complexity

```text
O(N)
```

Every node is visited once.

Copying valid paths costs extra, so practical complexity is:

```text
O(N + K×L)
```

where:

* `K` = number of valid paths
* `L` = average path length

---

### Space Complexity

```text
O(H)
```

Recursion stack + current path.

* Balanced Tree → `O(log N)`
* Skewed Tree → `O(N)`

Additional space is needed for storing output paths.
