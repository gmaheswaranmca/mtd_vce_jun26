# LeetCode 129 - Sum Root to Leaf Numbers

## Category

Binary Tree, DFS

---

## Description

You are given the `root` of a binary tree containing digits from `0` to `9`.

Each root-to-leaf path represents a number.

For example:

```text
1 → 2 → 3
```

represents the number:

```text
123
```

Return the **total sum of all root-to-leaf numbers**.

---

## Sample Input

```text
Tree:

      1
     / \
    2   3
```

## Sample Output

```text
25
```

---

## Explanation

Root-to-leaf paths:

```text
1 → 2 = 12
1 → 3 = 13
```

Total:

```text
12 + 13 = 25
```

---

## Sample Input 2

```text
Tree:

        4
       / \
      9   0
     / \
    5   1
```

## Sample Output 2

```text
1026
```

---

## Explanation

```text
4→9→5 = 495
4→9→1 = 491
4→0   = 40

495 + 491 + 40 = 1026
```

---

# Idea to Solve

While moving from root to leaf:

Build the current number.

For every node:

```text
currentNumber = currentNumber * 10 + node.value
```

When a leaf node is reached:

```text
Add currentNumber to answer
```

Use DFS traversal.

# Algorithm

1. Start DFS from root.
2. Maintain current number.
3. Update:

```text
current = current * 10 + node.val
```

4. If node is leaf:

   * Return current.
5. Recursively calculate:

   * leftSum
   * rightSum
6. Return:

```text
leftSum + rightSum
```

---

# Pseudocode

```text
FUNCTION dfs(node, current)
    IF node IS NULL
        RETURN 0
    current = current * 10 + node.value
    IF node IS LEAF
        RETURN current
    leftSum = dfs(node.left, current)
    rightSum = dfs(node.right, current)
    RETURN leftSum + rightSum
END FUNCTION
FUNCTION sumNumbers(root)
    RETURN dfs(root, 0)
END FUNCTION
```

# Python

```python
class Solution:

    def dfs(self, node, current):

        if node is None:
            return 0

        current = current * 10 + node.val

        if node.left is None and node.right is None:
            return current

        leftSum = self.dfs(node.left, current)
        rightSum = self.dfs(node.right, current)

        return leftSum + rightSum

    def sumNumbers(self, root):

        return self.dfs(root, 0)
```
