# LeetCode 106 - Construct Binary Tree from Inorder and Postorder Traversal

## Category

Binary Tree, Divide and Conquer, Recursion

---

# Description

Given two integer arrays:

* `inorder`
* `postorder`

Construct and return the binary tree.

You may assume:

* All values are unique.
* The given traversals represent a valid binary tree.

---

## Traversal Properties

### Inorder

```text
Left → Root → Right
```

### Postorder

```text
Left → Right → Root
```

---

## Sample Input

```text
inorder   = [9,3,15,20,7]

postorder = [9,15,7,20,3]
```

---

## Sample Output

```text
        3
       / \
      9  20
         / \
        15  7
```

# Important Difference from 105

In Problem 105:

```text
Root
→ Build Left
→ Build Right
```

because preorder is:

```text
Root Left Right
```

---

In Problem 106:

We process postorder from right to left:

```text
Root Right Left
```

Therefore:

```text
Root
→ Build Right
→ Build Left
```

⚠️ This is the most important interview point.

# Idea to Solve

1. Create HashMap from inorder values to indices.
2. Maintain postorder index starting from end.
3. Create root from current postorder value.
4. Find root position in inorder.
5. Build right subtree first.
6. Build left subtree.
7. Return root.

---

# Algorithm

1. Build hashmap.
2. Initialize:

```text
postIndex = n-1
```

3. Create root using postorder[postIndex].
4. Decrement index.
5. Find root position in inorder.
6. Build:

   * right subtree
   * left subtree
7. Return root.

---

# Pseudocode

```text
GLOBAL postIndex

FUNCTION build(postorder, inorderMap, left, right)
    IF left > right
        RETURN NULL
    value = postorder[postIndex]
    postIndex = postIndex - 1
    root = new Node(value)
    mid = inorderMap[value]
    root.right = build(postorder, inorderMap, mid + 1, right)
    root.left = build(postorder, inorderMap, left, mid - 1)
    RETURN root
END FUNCTION
FUNCTION buildTree(inorder, postorder)
    inorderMap = {}
    STORE all indices
    postIndex = length(postorder)-1
    RETURN build(postorder, inorderMap, 0, length(inorder)-1)
END FUNCTION
```

# Python

```python
class Solution:

    def build(self,
              postorder,
              inorderMap,
              left,
              right):

        if left > right:
            return None

        value = postorder[self.postIndex]
        self.postIndex -= 1

        root = TreeNode(value)

        mid = inorderMap[value]

        root.right = self.build(
            postorder,
            inorderMap,
            mid + 1,
            right
        )

        root.left = self.build(
            postorder,
            inorderMap,
            left,
            mid - 1
        )

        return root

    def buildTree(self,
                  inorder,
                  postorder):

        inorderMap = {}

        for i, value in enumerate(inorder):
            inorderMap[value] = i

        self.postIndex = len(postorder) - 1

        return self.build(
            postorder,
            inorderMap,
            0,
            len(inorder) - 1
        )
```

# C++

```cpp
class Solution {
public:

    int postIndex;

    TreeNode* build(
        vector<int>& postorder,
        unordered_map<int,int>& inorderMap,
        int left,
        int right) {

        if(left > right)
            return nullptr;

        int value = postorder[postIndex--];

        TreeNode* root = new TreeNode(value);

        int mid = inorderMap[value];

        root->right =
            build(postorder,
                  inorderMap,
                  mid + 1,
                  right);

        root->left =
            build(postorder,
                  inorderMap,
                  left,
                  mid - 1);

        return root;
    }

    TreeNode* buildTree(
        vector<int>& inorder,
        vector<int>& postorder) {

        unordered_map<int,int> inorderMap;

        for(int i=0;i<inorder.size();i++)
            inorderMap[inorder[i]] = i;

        postIndex = postorder.size()-1;

        return build(
            postorder,
            inorderMap,
            0,
            inorder.size()-1
        );
    }
};
```

# Java

```java
class Solution {

    private int postIndex;

    private TreeNode build(
            int[] postorder,
            Map<Integer,Integer> inorderMap,
            int left,
            int right) {

        if(left > right)
            return null;

        int value = postorder[postIndex--];

        TreeNode root = new TreeNode(value);

        int mid = inorderMap.get(value);

        root.right =
            build(postorder,
                  inorderMap,
                  mid + 1,
                  right);

        root.left =
            build(postorder,
                  inorderMap,
                  left,
                  mid - 1);

        return root;
    }

    public TreeNode buildTree(
            int[] inorder,
            int[] postorder) {

        Map<Integer,Integer> inorderMap =
            new HashMap<>();

        for(int i=0;i<inorder.length;i++)
            inorderMap.put(inorder[i], i);

        postIndex = postorder.length - 1;

        return build(
            postorder,
            inorderMap,
            0,
            inorder.length - 1
        );
    }
}
```

# Interview Summary

| Problem | Traversals Given    | Root Position     | Build Order  |
| ------- | ------------------- | ----------------- | ------------ |
| 105     | Preorder + Inorder  | First in Preorder | Left → Right |
| 106     | Inorder + Postorder | Last in Postorder | Right → Left |

```text
105:
Root Left Right

106:
Left Right Root
```

Remember:

```text
Preorder → build Left first

Postorder → build Right first
```
