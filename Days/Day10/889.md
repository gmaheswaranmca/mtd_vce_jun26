# LeetCode 889 - Construct Binary Tree from Preorder and Postorder Traversal

## Category

Binary Tree, Divide & Conquer, Recursion

---

# Description

Given two integer arrays:

* `preorder`
* `postorder`

where:

```text
Preorder  = Root → Left → Right
Postorder = Left → Right → Root
```

Construct and return **any binary tree** that matches both traversals.

If multiple answers exist, return any one of them.

---

## Sample Input

```text
preorder  = [1,2,4,5,3,6,7]

postorder = [4,5,2,6,7,3,1]
```

## Sample Output

```text
          1
        /   \
       2     3
      / \   / \
     4  5  6   7
```

---

# Important Observation

Unlike Problems 105 and 106:

```text
Preorder + Inorder
Inorder + Postorder
```

give a unique tree.

But:

```text
Preorder + Postorder
```

does **not always produce a unique tree**.

Example:

```text
Preorder  = [1,2]
Postorder = [2,1]
```

Both trees are valid:

```text
    1          1
   /            \
  2              2
```

Hence LeetCode says:

```text
Return ANY valid tree.
```

---

# Key Idea

### Preorder

```text
Root Left Right
```

First element is root.

```text
preorder[preIndex]
```

---

### Next preorder element

```text
preorder[preIndex + 1]
```

must be root of left subtree.

Suppose:

```text
preorder

[1,2,4,5,3,6,7]
   ^
 Left subtree root
```

Find this value inside postorder.

```text
postorder

[4,5,2,6,7,3,1]
     ^
```

Everything before it belongs to left subtree.

This tells us the left subtree size.

# Idea to Solve

For every recursive call:

1. Create root using current preorder element.
2. If subtree has one node:

   * Return root.
3. Next preorder value is left subtree root.
4. Find it in postorder.
5. Calculate left subtree size.
6. Recursively construct:

   * left subtree
   * right subtree
7. Return root.

---

# Algorithm

1. Build hashmap:

```text
value → index in postorder
```

2. Root = preorder[preStart].
3. Find left subtree root.
4. Compute left subtree size.
5. Construct left subtree.
6. Construct right subtree.
7. Return root.

---

# Pseudocode

```text
FUNCTION build(preStart, preEnd, postStart, postEnd)
    IF preStart > preEnd
        RETURN NULL
    root = new Node(preorder[preStart])
    IF preStart == preEnd
        RETURN root
    leftRoot = preorder[preStart + 1]
    index = postMap[leftRoot]
    leftSize = index - postStart + 1
    root.left = build(preStart + 1, preStart + leftSize,  postStart, index)
    root.right = build(preStart + leftSize + 1, preEnd, index + 1, postEnd - 1)
    RETURN root
END FUNCTION
```

# Python

```python
class Solution:

    def build(self,
              preorder,
              postorder,
              postMap,
              preStart,
              preEnd,
              postStart,
              postEnd):

        if preStart > preEnd:
            return None

        root = TreeNode(preorder[preStart])

        if preStart == preEnd:
            return root

        leftRoot = preorder[preStart + 1]

        index = postMap[leftRoot]

        leftSize = index - postStart + 1

        root.left = self.build(
            preorder,
            postorder,
            postMap,
            preStart + 1,
            preStart + leftSize,
            postStart,
            index
        )

        root.right = self.build(
            preorder,
            postorder,
            postMap,
            preStart + leftSize + 1,
            preEnd,
            index + 1,
            postEnd - 1
        )

        return root

    def constructFromPrePost(
            self,
            preorder,
            postorder):

        postMap = {}

        for i, value in enumerate(postorder):
            postMap[value] = i

        return self.build(
            preorder,
            postorder,
            postMap,
            0,
            len(preorder) - 1,
            0,
            len(postorder) - 1
        )
```

```cpp
class Solution {
public:

    unordered_map<int,int> postMap;

    TreeNode* build(
        vector<int>& preorder,
        vector<int>& postorder,
        int preStart,
        int preEnd,
        int postStart,
        int postEnd) {

        if(preStart > preEnd)
            return nullptr;

        TreeNode* root =
            new TreeNode(preorder[preStart]);

        if(preStart == preEnd)
            return root;

        int leftRoot =
            preorder[preStart + 1];

        int index =
            postMap[leftRoot];

        int leftSize =
            index - postStart + 1;

        root->left =
            build(preorder,
                  postorder,
                  preStart + 1,
                  preStart + leftSize,
                  postStart,
                  index);

        root->right =
            build(preorder,
                  postorder,
                  preStart + leftSize + 1,
                  preEnd,
                  index + 1,
                  postEnd - 1);

        return root;
    }

    TreeNode* constructFromPrePost(
            vector<int>& preorder,
            vector<int>& postorder) {

        for(int i=0;i<postorder.size();i++)
            postMap[postorder[i]] = i;

        return build(
            preorder,
            postorder,
            0,
            preorder.size()-1,
            0,
            postorder.size()-1
        );
    }
};
```

# Java

```java
class Solution {

    private Map<Integer,Integer> postMap =
        new HashMap<>();

    private TreeNode build(
            int[] preorder,
            int[] postorder,
            int preStart,
            int preEnd,
            int postStart,
            int postEnd) {

        if(preStart > preEnd)
            return null;

        TreeNode root =
            new TreeNode(preorder[preStart]);

        if(preStart == preEnd)
            return root;

        int leftRoot =
            preorder[preStart + 1];

        int index =
            postMap.get(leftRoot);

        int leftSize =
            index - postStart + 1;

        root.left =
            build(preorder,
                  postorder,
                  preStart + 1,
                  preStart + leftSize,
                  postStart,
                  index);

        root.right =
            build(preorder,
                  postorder,
                  preStart + leftSize + 1,
                  preEnd,
                  index + 1,
                  postEnd - 1);

        return root;
    }

    public TreeNode constructFromPrePost(
            int[] preorder,
            int[] postorder) {

        for(int i=0;i<postorder.length;i++)
            postMap.put(postorder[i], i);

        return build(
            preorder,
            postorder,
            0,
            preorder.length - 1,
            0,
            postorder.length - 1
        );
    }
}
```
