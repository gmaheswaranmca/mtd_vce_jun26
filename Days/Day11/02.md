# LeetCode 230 — Kth Smallest Element in a BST

## Problem Statement

Given the root of a Binary Search Tree (BST) and an integer `k`, return the **kth smallest value** in the BST.

---

## Example

### Input

```text
        5
       / \
      3   6
     / \
    2   4
   /
  1

k = 3
```

### Output

```text
3
```

---

# Key BST Insight

For a BST:

```text
Inorder Traversal
(Left → Root → Right)
```

always produces values in **sorted order**.

Example:

```text
        5
       / \
      3   6
     / \
    2   4
   /
  1
```

Inorder:

```text
1 2 3 4 5 6
```

3rd smallest = **3**

---

# Idea

Perform an inorder traversal.

Every time we visit a node:

```text
count = count + 1
```

When:

```text
count == k
```

that node is the answer.

---

# Algorithm

1. Initialize `count = 0`.
2. Perform inorder traversal.
3. Visit left subtree.
4. Increment count.
5. If count equals k:

   * Store answer.
   * Stop further processing.
6. Visit right subtree.
7. Return answer.

# Pseudocode

```text
count = 0
answer = -1
FUNCTION inorder(node)
    IF node IS NULL
        RETURN
    inorder(node.left)
    count = count + 1
    IF count == k
        answer = node.value
        RETURN
    inorder(node.right)
END FUNCTION
FUNCTION kthSmallest(root, k)
    inorder(root)
    RETURN answer
END FUNCTION
```

# Python

```python
class Solution:

    def kthSmallest(self, root, k):

        self.count = 0
        self.answer = -1

        def inorder(node):

            if node is None:
                return

            inorder(node.left)

            self.count += 1

            if self.count == k:
                self.answer = node.val
                return

            inorder(node.right)

        inorder(root)

        return self.answer
```

# Java

```java
class Solution {

    int count = 0;
    int answer = -1;

    private void inorder(TreeNode node, int k) {

        if (node == null)
            return;

        inorder(node.left, k);

        count++;

        if (count == k) {
            answer = node.val;
            return;
        }

        inorder(node.right, k);
    }

    public int kthSmallest(TreeNode root, int k) {

        inorder(root, k);

        return answer;
    }
}
```

            return;

        inorder(node->right, k);
