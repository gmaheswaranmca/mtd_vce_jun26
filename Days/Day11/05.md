# LeetCode 108 — Convert Sorted Array to Binary Search Tree

## Problem Statement

Given a sorted (ascending order) integer array `nums`, convert it into a **height-balanced BST**.

A height-balanced BST is a BST where the depths of the left and right subtrees differ by at most 1.

---

## Example

### Input

```text
nums = [-10,-3,0,5,9]
```

### Output

One possible BST:

```text
        0
      /   \
    -10    5
      \      \
      -3      9
```

Another valid balanced BST may also be accepted.

---

# Key Observation

The array is already sorted.

```text
[-10, -3, 0, 5, 9]
```

Middle element:

```text
0
```

becomes root.

Everything left of 0:

```text
[-10, -3]
```

must go into left subtree.

Everything right of 0:

```text
[5, 9]
```

must go into right subtree.

This naturally satisfies BST ordering.

---

# Idea

Choose the middle element as root.

Why middle?

Because it creates nearly equal-sized left and right subtrees.

Example:

```text
[-10,-3,0,5,9]

      0
```

Left half:

```text
[-10,-3]
```

Right half:

```text
[5,9]
```

Repeat the same process recursively.

# Algorithm

1. If left > right:

   * return NULL
2. Find middle index.
3. Create node using middle element.
4. Recursively build left subtree.
5. Recursively build right subtree.
6. Return root.

---

# Pseudocode

```text
FUNCTION build(nums, left, right)
    IF left > right
        RETURN NULL
    mid = (left + right) / 2
    root = new Node(nums[mid])
    root.left = build(nums, left, mid - 1)
    root.right = build(nums, mid + 1, right)
    RETURN root
END FUNCTION
FUNCTION sortedArrayToBST(nums)
    RETURN build(nums, 0, nums.length - 1)
END FUNCTION
```

# Python

```python
class Solution:

    def sortedArrayToBST(self, nums):

        def build(left, right):

            if left > right:
                return None

            mid = (left + right) // 2

            root = TreeNode(nums[mid])

            root.left = build(
                left,
                mid - 1
            )

            root.right = build(
                mid + 1,
                right
            )

            return root

        return build(
            0,
            len(nums) - 1
        )
```

        int[] nums,
        int left,
        int right
    ) {

        if (left > right)
            return null;

        int mid =
            (left + right) / 2;

            build(

                mid + 1,
                right

        return root;
    }

    public TreeNode sortedArrayToBST(

            0,

        );
    }
}

class Solution {
public:

    TreeNode* build(
        vector<int>& nums,
        int left,
        int right
    ) {

        if(left > right)
            return nullptr;

        int mid =
            (left + right) / 2;

        TreeNode* root =
            new TreeNode(nums[mid]);

        root->left =
            build(
                nums,
                left,
                mid - 1
            );

        root->right =
            build(
                nums,
                mid + 1,
                right
            );

        return root;
    }

    TreeNode* sortedArrayToBST(
        vector<int>& nums
    ) {

        return build(
            nums,
            0,
            nums.size() - 1
        );
    }
};
