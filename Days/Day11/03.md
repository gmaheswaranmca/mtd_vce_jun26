# LeetCode 701 — Insert into a Binary Search Tree

## Problem Statement

You are given the root of a BST and a value `val`.

Insert `val` into the BST and return the root of the BST.

The BST property must remain valid.

---

## Example

### Input

```text
        4
       / \
      2   7
     / \
    1   3

val = 5
```

### Output

```text
        4
       / \
      2   7
     / \ /
    1  3 5
```

---

# BST Property

```text
Left Subtree  < Root < Right Subtree
```

To insert a value:

* Smaller → Go Left
* Greater → Go Right
* Continue until NULL
* Create new node there

---

# Recursive Thinking

Suppose:

```text
insert(root, 5)
```

At node 4:

```text
root.right =
    insert(root.right, 5)
```

At node 7:

```text
root.left =
    insert(root.left, 5)
```

At NULL:

```text
return new Node(5)
```

Then recursion rebuilds links automatically.

---

# Recursion Tree

```text
insert(4,5)
|
+-- insert(7,5)
    |
    +-- insert(NULL,5)
        |
        +-- create Node(5)
```

Return Flow:

```text
Node(5)
  ↑
7.left = Node(5)
  ↑
4.right = 7
```

---

# Algorithm

1. If root is NULL:

   * Create new node.
2. If val < root.val:

   * Insert into left subtree.
3. Else:

   * Insert into right subtree.
4. Return root.

---

# Pseudocode

```text
FUNCTION insert(root, val)
    IF root IS NULL
        RETURN new Node(val)
    IF val < root.value
        root.left = insert(root.left, val)
    ELSE
        root.right = insert(root.right, val)
    RETURN root
END FUNCTION
```

# Python

```python
class Solution:

    def insertIntoBST(self, root, val):

        if root is None:
            return TreeNode(val)

        if val < root.val:
            root.left = self.insertIntoBST(
                root.left,
                val
            )
        else:
            root.right = self.insertIntoBST(
                root.right,
                val
            )

        return root
```

# Java

class Solution {
    public TreeNode insertIntoBST(TreeNode root,int val) {
        if (root == null)

        if (val < root.val)
            root.left = insertIntoBST(root.left, val);

            root.right = insertIntoBST(root.right, val);
        return root;
    }
}

# Iterative Version (Interview Friendly)

### Python

```python
class Solution:
    def insertIntoBST(self, root, val):
        if root is None:
            return TreeNode(val)
        curr = root
        while True:
            if val < curr.val:
                if curr.left is None:
                    curr.left = TreeNode(val)
                    break
                curr = curr.left
            else:
                if curr.right is None:
                    curr.right = TreeNode(val)
                    break
                curr = curr.right
        return root
```
