# LeetCode 207 — Course Schedule
```
The solution uses BFS Topological Sorting (Kahn's Algorithm). It stores the number of prerequisites of each course using an indegree array, repeatedly processes courses with indegree 0, removes their outgoing edges, and checks whether all courses can be processed. If all courses are processed, there is no cycle; otherwise, a cycle exists and finishing all courses is impossible.

Points to remember:
    Course Schedule
    Task Scheduling
    Dependency Resolution
    Build Order
    Package Installation Order
    ⇒ Topological Sort
    ⇒ Kahn's Algorithm (BFS)
    ⇒ Indegree Array + Queue
```
## Problem Statement

There are `numCourses` courses numbered from `0` to `numCourses - 1`.

Some courses have prerequisites.

Example:

```text id="7n36cl"
[1,0]
```

means:

```text id="4kh0o6"
To take course 1,
you must first complete course 0.
```

Return:

```text id="lm8rlv"
true
```

if all courses can be completed.

Otherwise:

```text id="ce6lm4"
false
```

```
class Solution:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        # Build adjacency list
        graph = [[] for _ in range(numCourses)]
        # indegree[i] = number of prerequisites required for course i
        indegree = [0] * numCourses
        # Build graph and indegree array
        for course, prereq in prerequisites:
            graph[prereq].append(course)  # prereq -> course
            indegree[course] += 1
        # Queue for courses having no prerequisites
        q = deque()
        # Add all courses with indegree 0
        for i in range(numCourses):
            if indegree[i] == 0:
                q.append(i)
        # Count of courses that can be completed
        completed = 0
        # Kahn's Algorithm (BFS Topological Sort)
        while q:
            # Take a course whose prerequisites are completed
            node = q.popleft()
            # Mark course as completed
            completed += 1
            # Remove its outgoing edges
            for neighbor in graph[node]:
                # One prerequisite is now satisfied
                indegree[neighbor] -= 1
                # If all prerequisites are satisfied,
                # the course is ready to be taken
                if indegree[neighbor] == 0:
                    q.append(neighbor)
        # If all courses are completed,
        # there is no cycle
        return completed == numCourses
```

# Algorithm

### Build Graph

For each prerequisite:

```text id="5jzdhw"
course, prereq
```

Add:

```text id="piyfrv"
prereq → course
```

Increase indegree.

---

### Kahn's Algorithm

Push all nodes having:

```text id="gqoqdy"
indegree = 0
```

into queue.

---

### BFS

While queue not empty:

```text id="v46dya"
remove node

count++

for each neighbor

    indegree--

    if indegree == 0
        push queue
```

---

### Final Check

If:

```text id="v5pjwi"
count == numCourses
```

return true.

Else:

```text id="jl7paj"
return false
```

---

# Pseudocode

```text id="c9o2ck"
Build graph
Build indegree array
Queue all nodes
with indegree 0
count = 0
WHILE queue not empty
    node = dequeue
    count++
    FOR each neighbor
        indegree--
        IF indegree == 0
            enqueue
IF count == numCourses
    RETURN TRUE
ELSE
    RETURN FALSE
```

```python
from collections import deque

class Solution:
    def canFinish(self, numCourses, prerequisites):
        graph = [[] for _ in range(numCourses)]
        indegree = [0] * numCourses
        for course, prereq in prerequisites:
            graph[prereq].append(course)
            indegree[course] += 1
        q = deque()
        for i in range(numCourses):
            if indegree[i] == 0:
                q.append(i)
        completed = 0
        while q:
            node = q.popleft()
            completed += 1
            for neighbor in graph[node]:
                indegree[neighbor] -= 1
                if indegree[neighbor] == 0:
                    q.append(neighbor)
        return completed == numCourses
```

# Java

```java
class Solution {

    public boolean canFinish(
        int numCourses,
        int[][] prerequisites
    ) {

        List<Integer>[] graph =
            new ArrayList[numCourses];

        for(int i=0;i<numCourses;i++)
            graph[i] = new ArrayList<>();

        int[] indegree =
            new int[numCourses];

        for(int[] p : prerequisites){

            int course = p[0];
            int prereq = p[1];

            graph[prereq].add(course);

            indegree[course]++;
        }

        Queue<Integer> q =
            new LinkedList<>();

        for(int i=0;i<numCourses;i++){

            if(indegree[i]==0)
                q.offer(i);
        }

        int completed = 0;

        while(!q.isEmpty()){

            int node = q.poll();

            completed++;

            for(int neighbor : graph[node]){

                indegree[neighbor]--;

                if(indegree[neighbor]==0)
                    q.offer(neighbor);
            }
        }

        return completed == numCourses;
    }
}
```

# C++

```cpp
class Solution {
public:

    bool canFinish(
        int numCourses,
        vector<vector<int>>& prerequisites
    ) {

        vector<vector<int>>
            graph(numCourses);

        vector<int>
            indegree(numCourses,0);

        for(auto &p : prerequisites){

            int course = p[0];
            int prereq = p[1];

            graph[prereq].push_back(course);

            indegree[course]++;
        }

        queue<int> q;

        for(int i=0;i<numCourses;i++){

            if(indegree[i]==0)
                q.push(i);
        }

        int completed = 0;

        while(!q.empty()){

            int node = q.front();
            q.pop();

            completed++;

            for(int neighbor : graph[node]){

                indegree[neighbor]--;

                if(indegree[neighbor]==0)
                    q.push(neighbor);
            }
        }

        return completed == numCourses;
    }
};
```
