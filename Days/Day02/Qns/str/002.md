# Longest Substring Without Repeating Characters

Find the length of the longest substring containing unique characters only.

Example:

```text id="1pmqg6"
Input:  s = "abcabcbb"
Output: 3
Explanation: "abc"

Input:  s = "bbbbb"
Output: 1
Explanation: "b"

Input:  s = "pwwkew"
Output: 3
Explanation: "wke"
```

* Expand right pointer.

O(2n) \approx O(n)

# Optimized Solution Comparison

| Pattern                  | Time  | Space | Best Use                    |
| ------------------------ | ----- | ----- | --------------------------- |
| Brute Force              | O(n³) | O(n)  | Understanding               |
| Sliding Window + Set     | O(n)  | O(n)  | Standard interview solution |
| Sliding Window + HashMap | O(n)  | O(n)  | Best practical approach     |
| ASCII Array Optimization | O(n)  | O(1)  | Fastest for ASCII strings   |

# Interview Pattern Recognition

| Clue in Question     | Pattern                |
| -------------------- | ---------------------- |
| longest substring    | Sliding Window         |
| unique characters    | HashSet / Frequency    |
| contiguous substring | Window                 |
| maximum length       | Expand + shrink window |
| repeating characters | Move left pointer      |
