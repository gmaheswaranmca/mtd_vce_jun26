# Longest Palindromic Substring

Find the longest substring that is a palindrome.

Example:

```text id="glq7a5"
Input:  s = "babad"
Output: "bab"
Explanation: "aba" is also valid.

Input:  s = "cbbd"
Output: "bb"
```

# Pattern 2 — Dynamic Programming Pattern

## Idea

* Use DP table.
* `dp[i][j] = True` if substring `s[i:j+1]` is palindrome.

Rules:

* Single char → palindrome
* Two chars same → palindrome
* Inner substring palindrome + outer chars same

## Pattern Type

* Dynamic Programming
* Interval DP

## Time Complexity

[
O(n^2)
]

[
O(n^2)
]

# Pattern 3 — Expand Around Center Pattern

## Idea

Every palindrome expands from center.

Two centers possible:

* Odd length → single center
* Even length → double center

Expand outward while characters match.

* Two Pointer
