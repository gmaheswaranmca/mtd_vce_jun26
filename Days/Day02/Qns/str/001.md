# Valid Anagram

Check whether two strings contain the same characters with the same frequency.

Example:

```text
Input:  s = "anagram", t = "nagaram"
Output: True

Input:  s = "rat", t = "car"
Output: False
```

---

# Pattern 1 — Sorting Pattern

## Idea

* Sort both strings.
* If sorted strings are equal → anagram.

## Pattern Type

* Sorting
* Comparison

## Time Complexity

[
O(n \log n)
]

## Code

```python
def isAnagram(s, t):
    # Code Here


# Test
print(isAnagram("anagram", "nagaram"))  # True
print(isAnagram("rat", "car"))          # False
```

# Optimized Solution Comparison

| Pattern           | Time       | Space | Best Use                             |
| ----------------- | ---------- | ----- | ------------------------------------ |
| Sorting           | O(n log n) | O(n)  | Simple interviews                    |
| HashMap Frequency | O(n)       | O(n)  | General best solution                |
| Fixed Array Count | O(n)       | O(1)  | Best optimized for lowercase letters |
| Counter           | O(n)       | O(n)  | Pythonic short solution              |
