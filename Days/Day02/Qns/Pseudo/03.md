# Valid Anagram

**Problem:** Check whether two strings contain the same characters with the same frequency.

**Companies:** Amazon, Google, Microsoft, Meta, Adobe

**Difficulty:** Easy

---

# Solution Patterns

## Pattern 1: Sorting

### Idea

Two strings are anagrams if their sorted forms are identical.

### Time Complexity

* O(N log N)

### Space Complexity

* O(N)

### Pseudocode

```text
FUNCTION IsAnagram(s, t)

    IF LENGTH(s) ≠ LENGTH(t)
        RETURN FALSE

    sortedS = SORT(s)
    sortedT = SORT(t)

    RETURN sortedS = sortedT
```

---

## Pattern 2: Two Frequency Arrays (Lowercase English Letters)

### Idea

Count occurrences of each character in both strings and compare frequencies.

### Time Complexity

* O(N)

### Space Complexity

* O(1)
  (26 lowercase letters)

### Pseudocode

```text
FUNCTION IsAnagram(s, t)

    IF LENGTH(s) ≠ LENGTH(t)
        RETURN FALSE

    countS[26] = {0}
    countT[26] = {0}

    FOR EACH ch IN s
        countS[ch - 'a']++

    FOR EACH ch IN t
        countT[ch - 'a']++

    FOR i = 0 TO 25

        IF countS[i] ≠ countT[i]
            RETURN FALSE

    RETURN TRUE
```

---

## Pattern 3: Hash Map Frequency Count

### Idea

Store character frequencies using a hash map.

Works for:

* Lowercase letters
* Uppercase letters
* Digits
* Unicode characters

### Time Complexity

* O(N)

### Space Complexity

* O(K)

K = Number of unique characters

### Pseudocode

```text
FUNCTION IsAnagram(s, t)

    IF LENGTH(s) ≠ LENGTH(t)
        RETURN FALSE

    freq = EMPTY_MAP

    FOR EACH ch IN s
        freq[ch] = freq.get(ch, 0) + 1

    FOR EACH ch IN t

        IF ch NOT IN freq
            RETURN FALSE

        freq[ch] = freq[ch] - 1

        IF freq[ch] < 0
            RETURN FALSE

    RETURN TRUE
```

---

## Pattern 4: Single Hash Map Increment-Decrement

### Idea

Increase counts using first string and decrease using second string simultaneously.

After processing, all counts must become zero.

### Time Complexity

* O(N)

### Space Complexity

* O(K)

### Pseudocode

```text
FUNCTION IsAnagram(s, t)

    IF LENGTH(s) ≠ LENGTH(t)
        RETURN FALSE

    freq = EMPTY_MAP

    FOR i = 0 TO LENGTH(s)-1

        freq[s[i]] = freq.get(s[i], 0) + 1
        freq[t[i]] = freq.get(t[i], 0) - 1

    FOR EACH value IN freq

        IF value ≠ 0
            RETURN FALSE

    RETURN TRUE
```

# Pattern Comparison

| Pattern          | Approach                         | Time       | Space |
| ---------------- | -------------------------------- | ---------- | ----- |
| Sorting          | Sort both strings and compare    | O(N log N) | O(N)  |
| Frequency Arrays | Fixed-size counting array        | O(N)       | O(1)  |
| Hash Map Count   | Count and compare frequencies    | O(N)       | O(K)  |
| Single Hash Map  | Increment and decrement together | O(N)       | O(K)  |

---

# Optimized Solution

### Pattern

**Frequency Counting**

* Use **Frequency Array** when input contains only lowercase English letters.
* Use **Hash Map** when character set is unknown or Unicode is allowed.

### Why Optimal?

* Linear scan.
* No sorting required.
* Direct frequency comparison.
* Common interview expectation.

### Final Complexity

#### Lowercase English Letters

```text
Time  : O(N)
Space : O(1)
```

#### General Character Set

```text
Time  : O(N)
Space : O(K)
```
