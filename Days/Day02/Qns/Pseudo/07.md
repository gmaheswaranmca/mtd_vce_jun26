# Maximum Subarray

**Problem:** Find the contiguous subarray with the maximum possible sum.

**Companies:** Amazon, Microsoft, LinkedIn

**Difficulty:** Medium

---

# Solution Patterns

## Pattern 1: Brute Force (Generate All Subarrays)

### Idea

Generate every possible subarray, calculate its sum, and track the maximum.

### Time Complexity

* O(N³)

### Space Complexity

* O(1)

### Pseudocode

```text
FUNCTION MaximumSubarray(nums)

    maxSum = -∞

    FOR i = 0 TO n-1

        FOR j = i TO n-1

            currentSum = 0

            FOR k = i TO j
                currentSum = currentSum + nums[k]

            maxSum = MAX(maxSum, currentSum)

    RETURN maxSum
```

---

## Pattern 2: Improved Brute Force (Running Sum)

### Idea

Instead of recalculating every subarray sum, extend the subarray and keep a running sum.

### Time Complexity

* O(N²)

### Space Complexity

* O(1)

### Pseudocode

```text
FUNCTION MaximumSubarray(nums)

    maxSum = -∞

    FOR i = 0 TO n-1

        currentSum = 0

        FOR j = i TO n-1

            currentSum = currentSum + nums[j]

            maxSum = MAX(maxSum, currentSum)

    RETURN maxSum
```

---

## Pattern 3: Dynamic Programming (Kadane's Algorithm)

### Idea

At each position:

* Either extend the previous subarray.
* Or start a new subarray.

Recurrence:

dp[i]=\max(nums[i],\ dp[i-1]+nums[i])

### Time Complexity

* O(N)

### Space Complexity

* O(N)

### Pseudocode

```text
FUNCTION MaximumSubarray(nums)

    n = LENGTH(nums)

    dp[0] = nums[0]

    maxSum = dp[0]

    FOR i = 1 TO n-1

        dp[i] = MAX(nums[i],
                    dp[i-1] + nums[i])

        maxSum = MAX(maxSum, dp[i])

    RETURN maxSum
```

---

## Pattern 4: Kadane's Algorithm (Space Optimized)

### Idea

Only the previous DP value is required.

Maintain:

* currentSum = best subarray ending at current index
* maxSum = global maximum

### Time Complexity

* O(N)

### Space Complexity

* O(1)

### Pseudocode

```text
FUNCTION MaximumSubarray(nums)

    currentSum = nums[0]
    maxSum = nums[0]

    FOR i = 1 TO n-1

        currentSum = MAX(nums[i],
                         currentSum + nums[i])

        maxSum = MAX(maxSum,
                     currentSum)

    RETURN maxSum
```

# Dry Run

Input:

```text
nums = [-2,1,-3,4,-1,2,1,-5,4]
```

| Element | currentSum | maxSum |
| ------- | ---------- | ------ |
| -2      | -2         | -2     |
| 1       | 1          | 1      |
| -3      | -2         | 1      |
| 4       | 4          | 4      |
| -1      | 3          | 4      |
| 2       | 5          | 5      |
| 1       | 6          | 6      |
| -5      | 1          | 6      |
| 4       | 5          | 6      |

Answer:

```text
6
```

Subarray:

```text
[4, -1, 2, 1]
```

# Pattern Comparison

| Pattern            | Approach                            | Time  | Space |
| ------------------ | ----------------------------------- | ----- | ----- |
| Brute Force        | Generate all subarrays              | O(N³) | O(1)  |
| Running Sum        | Incremental subarray sums           | O(N²) | O(1)  |
| DP Array           | Store best sum ending at each index | O(N)  | O(N)  |
| Kadane's Algorithm | DP with constant memory             | O(N)  | O(1)  |

---

# Optimized Solution

### Pattern

**Kadane's Algorithm**

### Why Optimal?

* Single traversal.
* No extra array required.
* Handles negative numbers naturally.
* Most commonly expected in coding interviews.

### Final Complexity

```text
Time  : O(N)
Space : O(1)
```

### Interview Pattern Recognition

Use **Kadane's Algorithm** when you see:

* Maximum/Minimum contiguous subarray
* Largest sum segment
* Best profit segment
* Continuous range optimization
* "Maximum sum of consecutive elements"

This is the standard optimal solution for the **Maximum Subarray** problem.
