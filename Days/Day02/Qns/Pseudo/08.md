# Longest Palindromic Substring

**Problem:** Find the longest contiguous substring that is a palindrome.

**Companies:** Amazon, Google, Microsoft, Bloomberg, Uber

**Difficulty:** Medium

---

# Solution Patterns

## Pattern 1: Brute Force (Generate All Substrings)

### Idea

Generate every possible substring and check whether it is a palindrome.

### Time Complexity

* O(N³)

### Space Complexity

* O(1)

### Pseudocode

```text
FUNCTION LongestPalindrome(s)

    longest = ""

    FOR start = 0 TO LENGTH(s)-1

        FOR end = start TO LENGTH(s)-1

            substring = s[start...end]

            IF IsPalindrome(substring)

                IF LENGTH(substring) > LENGTH(longest)
                    longest = substring

    RETURN longest


FUNCTION IsPalindrome(str)

    left = 0
    right = LENGTH(str)-1

    WHILE left < right

        IF str[left] ≠ str[right]
            RETURN FALSE

        left++
        right--

    RETURN TRUE
```

## Pattern 2: Dynamic Programming

### Idea

Define:

dp[i][j]=1\ \text{if}\ s[i]=s[j]\ \text{and}\ (j-i<3\ \text{or}\ dp[i+1][j-1])

`dp[i][j]` indicates whether substring `s[i...j]` is a palindrome.

### Time Complexity

* O(N²)

### Space Complexity

* O(N²)

### Pseudocode

```text
FUNCTION LongestPalindrome(s)

    n = LENGTH(s)

    dp[n][n] = FALSE

    start = 0
    maxLen = 1

    FOR i = 0 TO n-1
        dp[i][i] = TRUE

    FOR len = 2 TO n

        FOR i = 0 TO n-len

            j = i + len - 1

            IF s[i] = s[j]

                IF len = 2 OR len = 3
                    dp[i][j] = TRUE

                ELSE
                    dp[i][j] = dp[i+1][j-1]

            ELSE
                dp[i][j] = FALSE

            IF dp[i][j] = TRUE AND len > maxLen

                start = i
                maxLen = len

    RETURN SUBSTRING(s, start, maxLen)
```

## Pattern 3: Expand Around Center

### Idea

Every palindrome has a center.

For each index:

* Expand for odd-length palindrome
* Expand for even-length palindrome

Keep track of the longest palindrome found.

### Time Complexity

* O(N²)

### Space Complexity

* O(1)

### Pseudocode

```text
FUNCTION LongestPalindrome(s)

    IF LENGTH(s) <= 1
        RETURN s

    start = 0
    end = 0

    FOR i = 0 TO LENGTH(s)-1

        len1 = Expand(s, i, i)
        len2 = Expand(s, i, i+1)

        currentLen = MAX(len1, len2)

        IF currentLen > end - start + 1

            start = i - (currentLen - 1) / 2
            end = i + currentLen / 2

    RETURN SUBSTRING(s, start, end-start+1)


FUNCTION Expand(s, left, right)

    WHILE left >= 0 AND
          right < LENGTH(s) AND
          s[left] = s[right]

        left--
        right++

    RETURN right - left - 1
```

## Pattern 4: Manacher's Algorithm

### Idea

Transform the string by inserting separators.

Example:

```text
"abba" → "#a#b#b#a#"
```

Reuse previously computed palindrome radii to avoid redundant expansion.

### Time Complexity

* O(N)

### Space Complexity

* O(N)

### Pseudocode

```text
FUNCTION LongestPalindrome(s)

    transformed = AddSeparators(s)

    center = 0
    right = 0

    p[] = ARRAY OF ZEROS

    FOR i = 0 TO LENGTH(transformed)-1

        mirror = 2*center - i

        IF i < right
            p[i] = MIN(right-i, p[mirror])

        WHILE CharactersMatchAroundCenter(i)

            p[i]++

        IF i + p[i] > right

            center = i
            right = i + p[i]

    RETURN ExtractLongestPalindrome()
```

# Pattern Comparison

| Pattern              | Approach                     | Time  | Space |
| -------------------- | ---------------------------- | ----- | ----- |
| Brute Force          | Check all substrings         | O(N³) | O(1)  |
| Dynamic Programming  | Store palindrome states      | O(N²) | O(N²) |
| Expand Around Center | Expand from every center     | O(N²) | O(1)  |
| Manacher's Algorithm | Reuse palindrome information | O(N)  | O(N)  |

# Optimized Solution

## Interview-Preferred Solution

### Pattern

**Expand Around Center**

### Why Preferred?

* Much easier to implement than Manacher's.
* Uses constant extra space.
* Accepted comfortably for typical constraints.
* Commonly expected in interviews.

### Complexity

```text
Time  : O(N²)
Space : O(1)
```

## Theoretical Best Solution

### Pattern

**Manacher's Algorithm**

### Complexity

```text
Time  : O(N)

```

### Drawback

* Complex implementation.

# Pattern Recognition

Use **Expand Around Center** when you see:

* Longest palindromic substring
* Count palindromic substrings
* Palindrome expansion
* Symmetric character matching
* Continuous palindrome range problems

Use **Manacher's Algorithm** when:

* Input size is very large.
* Linear-time palindrome processing is required.
* Interview explicitly asks for the most optimal solution.

### Recommended Answer in Interviews

```text
Pattern: Expand Around Center
Time   : O(N²)
Space  : O(1)
```

This is the most commonly expected solution for **Longest Palindromic Substring**.
