# Two Sum

Given an array of integers `nums` and an integer `target`, return the indices of the two numbers such that they add up to `target`.

Example:

```python
nums = [2,7,11,15]
target = 9

Output: [0,1]
```

---

# Solution Patterns

## 1. Brute Force Pattern

### Pattern Used

* Nested Loop
* Compare every pair

### Idea

Check all possible pairs and see whether:

```python
nums[i] + nums[j] == target
```

### Time Complexity

* Time: `O(n²)`
* Space: `O(1)`

### Code

```python
def two_sum_bruteforce(nums, target):
    n = len(nums)

    for i in range(n):
        for j in range(i + 1, n):

            if nums[i] + nums[j] == target:
                return [i, j]

    return []
```

### Execution

```python
nums = [2,7,11,15]
target = 9

print(two_sum_bruteforce(nums, target))
```

Output:

```python
[0, 1]
```

# 2. Hashing Pattern (Optimized)

### Pattern Used

* Hash Map / Dictionary
* Lookup Optimization

### Main Idea

Instead of checking every pair:

* Store visited numbers in a dictionary
* For every number:

Need:

```python
target - current_number
```

If already present in dictionary:

* pair found

---

## Visual Understanding

Suppose:

```python

target = 9
```

Current Number = `7`

Need:

```python
9 - 7 = 2
```

If `2` already exists in hashmap:

```python
{2:0}
```

Then:

```python
2 + 7 = 9
```

Answer found.

# 3. Two Pointer Pattern (When Array is Sorted)

### Pattern Used

* Two Pointer Technique

### Important

Works efficiently only when array is sorted.

---

## Idea

* One pointer at beginning
* One pointer at end

If sum is:

* smaller → move left pointer
* larger → move right pointer

# Pattern Comparison

| Pattern     | Core Concept           | Time    | Space  | Best Use Case         |
| ----------- | ---------------------- | ------- | ------ | --------------------- |
| Brute Force | Check all pairs        | `O(n²)` | `O(1)` | Small arrays          |
| Hashing     | Store visited elements | `O(n)`  | `O(n)` | Best general solution |
| Two Pointer | Move from both ends    | `O(n)`  | `O(1)` | Sorted arrays         |
