# AVL Tree Learning Order

AVL Trees are a natural progression after:

```text
Binary Tree
↓
Binary Search Tree (BST)
↓
AVL Tree
↓
Red-Black Tree
↓
B-Tree / B+ Tree
```

# Phase 2: AVL Fundamentals

Learn:

## Height

```text
height(node)
=
1 + max(leftHeight, rightHeight)
```

## Balance Factor

```text
BF = Height(Left) - Height(Right)
```

Valid AVL:

```text
BF ∈ {-1, 0, +1}
```

### Most Important Problem

**1382 - Balance a Binary Search Tree**

Balance a Binary Search Tree

This is the closest LeetCode problem to understanding the purpose of AVL Trees: maintaining a balanced BST to guarantee **O(log n)** search, insertion, and deletion.
