# Trie Learning Order — LeetCode Number + Title

Trie is a relatively small topic. Learn it in this order:

**Basic Trie → Prefix Search → Dictionary Search → Backtracking + Trie → Advanced Trie Applications**

---

1. **208** - Implement Trie (Prefix Tree)
2. **1804** - Implement Trie II (Prefix Tree)

* Insert
* Search

* Prefix Count

4. **648** - Replace Words
5. **820** - Short Encoding of Words

**Learn**

6. **211** - Design Add and Search Words Data Structure
7. **676** - Implement Magic Dictionary
