# Backtracking Learning Order — LeetCode Number + Title

Backtracking is best learned in this order:

**Subsets → Combinations → Permutations → Combination Sum → String Backtracking → Grid Backtracking → Constraint Backtracking**

---

## Phase 1: Subsets

1. **78** - Subsets
2. **90** - Subsets II

**Learn**

* Include / Exclude
* Recursion Tree
* Handling Duplicates

---

## Phase 2: Combinations

3. **77** - Combinations
4. **216** - Combination Sum III

**Learn**

* Choose K elements
* Start Index Pattern

---

## Phase 3: Permutations

5. **46** - Permutations
6. **47** - Permutations II

**Learn**

* Used Array
* Duplicate Handling

---

## Phase 4: Combination Sum Family

7. **39** - Combination Sum
8. **40** - Combination Sum II
9. **377** - Combination Sum IV *(DP version later)*

**Learn**

* Reuse Elements
* No Reuse Elements
* Pruning

---

## Phase 5: String Backtracking

10. **17** - Letter Combinations of a Phone Number
11. **93** - Restore IP Addresses
12. **131** - Palindrome Partitioning

**Learn**

* Build String Recursively
* Partitioning

---

## Phase 6: Parentheses Backtracking

13. **22** - Generate Parentheses
14. **301** - Remove Invalid Parentheses

**Learn**

* Constraint Generation
* Validity Checks

---

## Phase 7: Grid Backtracking

15. **79** - Word Search
16. **212** - Word Search II

**Learn**

* DFS + Backtracking
* Visited Cells
* Trie Optimization

---

## Phase 8: Chess / Constraint Problems

17. **51** - N-Queens
18. **52** - N-Queens II

**Learn**

* Column Constraints
* Diagonal Constraints
* State Space Tree

---

## Phase 9: Advanced Partitioning

19. **698** - Partition to K Equal Sum Subsets
20. **473** - Matchsticks to Square

**Learn**

* Bucket Backtracking
* Symmetry Pruning

---

## Phase 10: Sudoku & Hard Backtracking

21. **37** - Sudoku Solver
22. **489** - Robot Room Cleaner

**Learn**

* Constraint Satisfaction
* Heavy Pruning
