### Beginner Level

700 → 701 → 938 → 530 → 783

Learn:

* BST property
* Search
* Insert
* Inorder traversal gives sorted order

### Intermediate Level

98 → 653 → 230 → 235 → 450

Learn:

* BST validation
* Kth smallest
* LCA using BST property
* Deletion cases

### Construction & Transformation

108 → 109 → 669 → 538 → 1038

Learn:

* Build BST
* Modify BST
* Reverse inorder techniques

### Advanced BST

173 → 99 → 285 → 510

Learn:

* Iterator design
* Recover corrupted BST
* Successor/Predecessor

### Dynamic Programming on BST

96 → 95

Learn:

* Catalan Numbers
* BST generation
* Tree DP

For interview preparation, the **must-solve BST set** is:

**700 → 701 → 98 → 230 → 235 → 450 → 108 → 173 → 99 → 96**

These 10 problems cover almost every fundamental BST concept.
