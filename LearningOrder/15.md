# Hashing Learning Order

## Phase 1: Hash Set Basics

1. **217** - Contains Duplicate
2. **219** - Contains Duplicate II
3. **349** - Intersection of Two Arrays
4. **202** - Happy Number

---

## Phase 2: Hash Map Basics

5. **1** - Two Sum
6. **205** - Isomorphic Strings
7. **290** - Word Pattern
8. **387** - First Unique Character in a String

---

## Phase 3: Frequency Counting

9. **242** - Valid Anagram
10. **383** - Ransom Note
11. **451** - Sort Characters By Frequency
12. **1207** - Unique Number of Occurrences

---

## Phase 4: Prefix Sum + Hash Map

13. **560** - Subarray Sum Equals K
14. **525** - Contiguous Array
15. **974** - Subarray Sums Divisible by K
16. **930** - Binary Subarrays With Sum

---

## Phase 5: Hashing + Strings

17. **49** - Group Anagrams
18. **438** - Find All Anagrams in a String
19. **3** - Longest Substring Without Repeating Characters
20. **128** - Longest Consecutive Sequence

---

## Phase 6: Advanced Hash Map

21. **36** - Valid Sudoku
22. **380** - Insert Delete GetRandom O(1)
23. **146** - LRU Cache
24. **981** - Time Based Key-Value Store

---

## Phase 7: Hard Hashing Problems

25. **76** - Minimum Window Substring
26. **30** - Substring with Concatenation of All Words
27. **149** - Max Points on a Line
28. **460** - LFU Cache

---

### Core Hashing Patterns

```text
Hash Set
↓
Hash Map
↓
Frequency Counting
↓
Prefix Sum + Hash Map
↓
String Hashing
↓
Sliding Window + Hash Map
↓
Design Problems (LRU/LFU)
```

If you master the first **15–20 problems** in this order, you'll cover the vast majority of hashing-based interview questions.
