# Greedy Learning Order — LeetCode Number + Title

Greedy is best learned after Arrays, Sorting, Intervals, and basic DP.

---

## Phase 1: Basic Greedy

1. **121** - Best Time to Buy and Sell Stock
2. **122** - Best Time to Buy and Sell Stock II
3. **55** - Jump Game
4. **45** - Jump Game II
5. **605** - Can Place Flowers

**Learn**

* Local optimum
* Greedy choice
* Maintaining best-so-far

---

## Phase 2: Sorting + Greedy

6. **455** - Assign Cookies
7. **561** - Array Partition
8. **506** - Relative Ranks
9. **135** - Candy
10. **1005** - Maximize Sum Of Array After K Negations

**Learn**

* Sort then greedily select
* Exchange arguments

---

## Phase 3: Interval Greedy

11. **435** - Non-overlapping Intervals
12. **452** - Minimum Number of Arrows to Burst Balloons
13. **646** - Maximum Length of Pair Chain
14. **1288** - Remove Covered Intervals
15. **1024** - Video Stitching

**Learn**

* Sort by end time
* Activity Selection
* Interval covering

---

## Phase 4: String Greedy

16. **316** - Remove Duplicate Letters
17. **1081** - Smallest Subsequence of Distinct Characters
18. **763** - Partition Labels
19. **1717** - Maximum Score From Removing Substrings

**Learn**

* Greedy with Stack
* Lexicographically smallest answers

---

## Phase 5: Heap + Greedy

20. **1642** - Furthest Building You Can Reach
21. **871** - Minimum Number of Refueling Stops
22. **2530** - Maximal Score After Applying K Operations
23. **1962** - Remove Stones to Minimize the Total

**Learn**

* Greedy + Priority Queue
* Choosing best resource usage

---

## Phase 6: Advanced Interval Greedy

24. **56** - Merge Intervals
25. **57** - Insert Interval
26. **763** - Partition Labels
27. **2406** - Divide Intervals Into Minimum Number of Groups

**Learn**

* Interval scheduling
* Sweep-line ideas

---

## Phase 7: Advanced Greedy

28. **134** - Gas Station
29. **649** - Dota2 Senate
30. **678** - Valid Parenthesis String
31. **738** - Monotone Increasing Digits
32. **659** - Split Array into Consecutive Subsequences

## Phase 8: Hard Greedy

33. **502** - IPO

35. **630** - Course Schedule III
